# WorldLines 에 기여하기

**Language:** [English](./CONTRIBUTING.md) · [简体中文](./CONTRIBUTING.zh.md) · [日本語](./CONTRIBUTING.ja.md) · [한국어](./CONTRIBUTING.ko.md)

> 상태: Preview — 기여 플로우는 프로젝트와 함께 진화합니다.

기여를 고려해 주셔서 감사합니다. WorldLines 는 공개 프리뷰 중이며, v0.1 의 기여 플로우는 의도적으로 가볍습니다.

## 시작 전

- **엔진 소스** (`neonrp`) 는 현재 클로즈드 프리뷰입니다. 이 저장소는 **설치 진입점, 문서, `stoneford` 스타터 템플릿** 을 배포합니다. 대부분의 커뮤니티 기여는 여기나 [`stoneford-worldlines`](https://github.com/LudicDynamics/stoneford-worldlines) 서브모듈에 들어가며, 엔진 바이너리에는 들어가지 않습니다.

## 환영하는 기여

| 영역                         | 기여 방법                                         |
|------------------------------|---------------------------------------------------|
| 문서 수정                    | 이 `*.md` 파일에 PR                               |
| 설치 경로 이슈               | OS / uv / Python 버전 포함해 issue                |
| Stoneford 컨텐츠             | `stoneford` 서브모듈 저장소에 PR                  |
| 새 템플릿                    | RFC issue 먼저, 동작하는 starter 와 함께 PR       |
| 엔진 버그 리포트             | 재현 단계와 `neonrp tui --verbose` 로그와 함께 issue |
| LLM provider 어댑터          | RFC issue 먼저 — 엔진 측 vs shim 조언합니다       |

## 워크플로우

1. 문제나 제안을 기술한 issue 를 엽니다.
2. 문서 / 템플릿: fork 후 PR.
3. 엔진 동작: 메인테이너가 범위를 확인할 때까지 코딩 대기 — 엔진 소스는 아직 비공개.
4. 모든 기여는 *무엇을* *왜* 바꿨는지에 대한 명확한 설명이 필요합니다.

## 스타일

- 문서: markdown, ASCII-safe 제목, emoji 없음.
- 명령 블록: 복사 붙여넣기 가능, 가짜 shell 프롬프트 없음.
- 주 언어는 영어. 필요하면 인라인 중국어 주석 OK.
- 생성 푸터 ("Generated by …" 등) 금지.

## 변경 테스트

문서만: 크로스링크 확인, `markdownlint` 있으면 돌리기.

템플릿 변경은 임시 디렉터리에서 검증:

```bash
cd /tmp && mkdir verify && cd verify
neonrp init
neonrp game new --template <your-template>
neonrp tui
```

## License Asymmetry (라이선스 비대칭성)

WorldLines 는 엔진과 데이터 사이에 의도적인 분리를 둡니다:

- **WorldLines 엔진** (`neonrp`): All rights reserved, 독점 프리뷰.
- **`templates/stoneford-worldlines/`**: AGPL-3.0-only.
- `stoneford-worldlines` 에 대한 기여는 AGPL-3.0 하에 라이선스됩니다.
- 기여는 엔진 코어에 대한 어떠한 권리도 부여하지 **않습니다**.
- stoneford 템플릿에 PR 을 제출하기 전에, 기여자는 자신의 작업을 AGPL-3.0 으로 라이선스할 수 있는지 확인해야 합니다.
