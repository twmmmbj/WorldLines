# 樱坂走廊 · Sakura Hallway

四月。樱花满开的坡道上，你遇到了一个女孩。

她站在坂道中央，抬头望着樱花。风吹过的时候，花瓣落在她的肩上，
她转过头来看你，说了一句话——你没能听清。

这是一个 Clannad 式的校园情感叙事世界。坡道、旧校舍、放学后的
走廊。没有战斗，没有任务。只有那些差点说出口的话，和一段还没开始
就已经让人想哭的故事。

**灵感来源：** Clannad、Kanon、Air 等 Key 社作品 —— 日常中的温暖与
遗憾，时间线与记忆的交错，人与人之间微小的羁绊。

## 看点

- **情感驱动的叙事 agent**：世界 agent 根据角色的好感度、季节和
  时间线推进故事，不是战斗系统，是"心动系统"。
- **记忆与回响**：每一句对话都被记住。角色会在很久以后提起你
  当初说过的一句话。
- **分支但不失控**：你可以选择靠近谁、说什么，但故事的基调——
  那个"樱花飘落的速度"——始终由 world-agent 保持。

## 世界观

一所建在坡道上的高中。四月，樱花满开。放学后的旧校舍走廊里
总是回荡着某个人的脚步声。据说，在樱花全部散落之前，在这条
走廊上向一个人说出真心话，那个愿望就会实现。

## 角色

- **小夜（Saya）** — 总是迟到、气喘吁吁跑上坡道的女孩。书包
  里装着画到一半的素描本。
- **文（Fumi）** — 图书室的常客。安静，但一开口就停不下来。
  在看一本没有结局的小说。
- **前辈（Senpai）** — 三年级的毕业生。即将离开，好像还有一句
  话没来得及说。
