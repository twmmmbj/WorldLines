# Dark Train — 暗夜列车

一辆永不停站的列车在没有风景的灰色里前行。你在第 6 节车厢醒来，
忘了自己是谁。前面还有五节车厢和一节连接走道，每节都把记忆收走一点。
没有人告诉你目的地。窗外只有雾，不动的、概念性的雾。

**Genre**: Quiet cosmic horror · Lovecraftian puzzle · Call of Cthulhu 7e

## 你扮演

一名调查者——但你已经不记得自己调查的是什么。
醒来时身上有一节断票根（"6 → 1"清晰可读，目的地被涂去）和一片金属
钥匙残片。无名、无证件、无电话。你的左无名指上有戒痕，戒指不在任何
口袋里。

## 怎么玩

```bash
# 在游戏目录里：
neonrp tui
# 选择 import → examples/dark-train
```

进入后：`@world-agent 开始游戏`

## 流程提示

- 每回合先看 HUD 行（车厢号 / HP / SAN / tick / 碎片数）
- 你的所有意图都允许尝试——荒诞或过分的会路由到判定，世界用一致的结果回答
- 检定全部 d100：1d100 ≤ 技能 = 成功；≤ 技能 / 2 = 困难成功；≤ 技能 / 5 = 极难成功
- SAN 检定 roll-under POW：低于当前 SAN = 损失少；等于或高于 = 损失多
- 单次 SAN 损失 ≥5 触发不定性疯狂检定（INT），失败留下后遗症（恐惧 / 妄想）

## 设计承诺

> 你永远不会赢得一场干净的胜利。每个谜题的解答都要付代价。
> 每个找回的真相，同时也是某个失去的东西。
