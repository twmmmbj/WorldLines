# D&D 5e 战斗速查(极简)

只覆盖这场遭遇战需要的部分。

## 回合流程
1. **先攻**:每个参战者掷 `d20 + 先攻调整值`,从高到低排序 = 本场行动顺序。
2. 轮到谁,谁做**一个动作**(这场里通常 = 一次攻击,或移动/躲藏)。
3. 所有人各行动一次 = 一轮(round)结束,进入下一轮,直到一方倒下。

## 攻击检定
- 掷 `d20 + 命中加值`,**≥ 目标 AC** → 命中。
- **暴击**:d20 骰面 = 20 → 必中,且**伤害骰翻倍**(调整值不翻倍)。
- **大失败**:d20 骰面 = 1 → 必失。
- 目标有半身掩护时 AC +2。

## 伤害
命中后掷武器伤害骰 + 调整值,从目标**当前 HP** 扣除。
布丽的偷袭额外 `+2d6`(满足条件时,每回合一次)。

## 倒下 / 结束
- **哥布林 HP ≤ 0** → 死亡,战斗结束,胜利。
- **冒险者 HP ≤ 0** → 倒地昏迷,退出本场战斗(本 demo 简化:不掷死亡豁免)。
- 哥布林**灵巧逃脱**成功逃进林子 → 战斗也结束(它跑了)。

## 给 DM 的提示
- 每次掷骰都交给 `referee`,把结果(命中/未中、伤害数)如实念给玩家,再写一句画面。
- 一次只推进**一个角色的行动**,然后停下等玩家;轮到哥布林时由你(DM)替它按 tactics 行动。
