# 心动系统(极简)

这个世界没有战斗、没有掷骰。唯一的「数值」是**好感度**,唯一的「计时器」是
**樱花飘落的速度**。

## 好感度(affection 0–100,每个角色各一份)
玩家每一句有意义的话、每一个靠近或回避的选择,都会让对象角色的好感度变化。
判定交给 `@heart`,它返回 delta(增减),world-agent 把新值写回
`game/npc/<id>.json`。

| 区间 | 等级(affection_level) | 表现 |
|---|---|---|
| 0–19 | 陌生 | 客气、有距离 |
| 20–39 | 在意 | 会主动找话题、记得你说过的小事 |
| 40–59 | 心动 | 在意你的反应、会脸红或回避目光 |
| 60–79 | 喜欢 | 愿意说出藏着的事(素描本 / 小说 / 那句话) |
| 80–99 | 想说出口 | 把话带到嘴边,却差一点 |
| 100 | 表白时刻 | 只要在旧校舍走廊说出真心话,愿望就会实现 |

## 樱花飘落(petals_remaining 100 → 0)
`run_state.petals_remaining` 是这段故事的截止线。每推进一天 / 一场重要场景,
world-agent 让它减少(约每天 −15~25)。**樱花散尽(归零)前没有走到「表白时刻」并
说出口,这段心动就停在差一点的地方**——这正是这个世界的基调。

## 记忆与回响
每个角色都有 `memory` 数组。玩家说过的、做过的有分量的事,world-agent 追加进去
(一句话 + 第几天)。角色会在很久以后主动提起:「你那天在坡道上说的那句……」
——**说过的话会回来**。

## 时间线(timeline)
每次进入故事可能落在不同的时间线节点(`first-meeting` / `after-school` /
`almost-spring-end` …)。同样的坡道,不同的时刻,遇见的人和能说的话都不同。

## 给 world-agent 的提示
- 每个有分量的玩家言行 → 调 `@heart` 判定 → 把新好感 + 这句话写回 npc 文件,
  并更新 run_state(天数 / 剩余花瓣 / 焦点角色)。
- 一次只推进**一拍**(一句对话 / 一个瞬间),然后停下,`AskUserQuestion` 给选项。
- 基调由你守住:温暖、留白、一点遗憾,「樱花飘落的速度」不要快进。
