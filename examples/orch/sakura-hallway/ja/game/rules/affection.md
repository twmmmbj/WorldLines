# ときめきシステム(超シンプル)

この世界には戦闘もなく、ダイスもない。唯一の「数値」は**好感度**、唯一の「タイマー」は
**桜が散る速さ**。

## 好感度(affection 0–100、キャラごとに一つずつ)
プレイヤーの意味ある一言、近づくか避けるかの一つ一つの選択が、対象キャラの好感度を変える。
判定は `@heart` に任せ、delta(増減)を返してもらい、world-agent が新しい値を
`game/npc/<id>.json` に書き戻す。

| 区間 | レベル(affection_level) | 様子 |
|---|---|---|
| 0–19 | 他人 | よそよそしく、距離がある |
| 20–39 | 気になる | 自分から話題を振る、きみの言った小さなことを覚えている |
| 40–59 | ときめき | きみの反応を気にする、頬を染めたり目をそらしたりする |
| 60–79 | 好き | 隠していたこと(スケッチブック / 小説 / あの一言)を打ち明けてくれる |
| 80–99 | 言いかけて | 言葉を口元まで持ってきて、あと少しで言えない |
| 100 | 告白の時 | 旧校舎の廊下で本心を口にしさえすれば、願いが叶う |

## 桜の散り(petals_remaining 100 → 0)
`run_state.petals_remaining` はこの物語の締め切り線。一日 / 一つの重要な場面を進めるごとに、
world-agent がこれを減らす(一日あたりおよそ −15~25)。**桜が散りきる(ゼロになる)前に「告白の時」まで辿り着いて
口にしなければ、このときめきはあと少しのところで止まる**——それがこの世界の基調。

## 記憶と響き
キャラはそれぞれ `memory` 配列を持つ。プレイヤーが言ったこと、したことのうち重みのあるものを、world-agent が追記する
(一言 + 何日目か)。キャラはずっと後になって自分から持ち出す:「あの日、坂道できみが言ったあの言葉……」
——**言った言葉は返ってくる**。

## タイムライン(timeline)
物語に入るたびに、異なるタイムラインの節目に落ちることがある(`first-meeting` / `after-school` /
`almost-spring-end` …)。同じ坂道でも、違う時刻なら、出会う人も、言える言葉も違う。

## world-agent へのヒント
- 重みのあるプレイヤーの言動ごと → `@heart` を呼んで判定 → 新しい好感 + この一言を npc ファイルに書き戻し、
  run_state(日数 / 残り花びら / 焦点キャラ)を更新する。
- 一度に進めるのは**一拍だけ**(一つのセリフ / 一つの瞬間)、それから止まって、`AskUserQuestion` で選択肢を出す。
- 基調はきみが守る:あたたかく、余白があり、ほんの少しの心残り、「桜が散る速さ」を早送りしない。
