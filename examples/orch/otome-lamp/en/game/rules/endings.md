# 命运抉择与结局

这段宫廷情劫有三处命运抉择(pivot)。world-agent 在对应夜数把抉择抛给玩家
(`AskUserQuestion`),玩家所选的 `flag` 追加进 `run_state.pivot_flags`。
三个 flag 集齐后,按下表 `requires` 组合落到八种结局之一。

## 三处抉择(pivot)

| pivot | 第几夜 | 处境 | 选项 A | 选项 B |
|---|---|---|---|---|
| pivot1 | 第 2 夜 | 你无意中听到太子与人密谋陷害二皇子 | `silent` 装作没听见,保全自己 | `warn` 偷偷提醒二皇子提防 |
| pivot2 | 第 4 夜 | 三皇子向你表白,但他可能在利用你接近引魂灯 | `accept` 接受这份心意,赌一把 | `refuse` 婉拒并保持距离,明哲保身 |
| pivot3 | 第 7 夜 | 太子以你家人性命要挟,逼你偷取引魂灯秘钥 | `yield` 屈服,交出秘钥 | `defy` 宁死不从,向皇上告发太子 |

> pivot 与情丝并行:玩家平日对四位皇子的情丝由 `@heart` 累积;pivot 则是
> 不可回避的立场抉择,决定大结局的走向与底色。world-agent 在每个 pivot 上
> 也应据所选 flag 给相关皇子的情丝与 `run_state.vq` 一次明显的偏移。

## 八种结局(集齐三 flag 后判定)

| 结局 | requires | 基调 | 收束 |
|---|---|---|---|
| 灯灭人散 | silent + refuse + yield | bad | 沉默、拒三皇子、屈服太子;秘钥被盗,引魂灯熄灭,你被逐出宫,孤苦一生。 |
| 暗潮汹涌 | silent + refuse + defy | bittersweet | 沉默、拒三皇子、却告发太子;太子倒台,你因知情不报被牵连,保命却再无人信。 |
| 孤灯相守 | silent + accept + yield | bad | 沉默、受三皇子、屈服太子;三皇子察觉你的背叛,离你而去,你守着空灯终老。 |
| 灯谜心计 | silent + accept + defy | bittersweet | 沉默、受三皇子、告发太子;三皇子借机上位,你成了他权谋的棋子,富贵而无真情。 |
| 侠义情长 | warn + refuse + yield | good | 告密、拒三皇子、屈服太子;二皇子感念你,虽一度失势,却与你共患难,真情流露。 |
| 昭雪沉冤 | warn + refuse + defy | good | 告密、拒三皇子、告发太子;太子阴谋败露,二皇子洗冤,你成功臣,与二皇子携手余生。 |
| 灯影迷离 | warn + accept + yield | weird | 告密、受三皇子、屈服太子;三皇子与太子联手,你周旋其中,引魂灯被毁,你失踪乱世。 |
| 双灯同辉 | warn + accept + defy | good | 告密、受三皇子、告发太子;三皇子与二皇子联手扳倒太子,你与三皇子真心相待,引魂灯重明。 |

## 与「情根深种」的关系
- 上表是**主线结局**(由三处 pivot 决定大局走向)。
- 若在灯火燃尽前,某位皇子情丝到达 **100(情根深种)**,且玩家在**禁苑引魂灯阁**
  对他说出真心话 → 在主线收束的同时,叠加该皇子的**定情**(灯火骤亮印证真心)。
- 若灯火先归零、无人走到情根深种 → 主线按 pivot_flags 收束,所有情丝皆停在
  「差一点」,这正是本作底色。

## 给 world-agent 的提示
- pivot 的 flag **只追加、不覆盖**;若玩家在某夜未触发对应 pivot,可在下一合适场景补上,
  但顺序仍是 1 → 2 → 3。
- 三 flag 集齐 → 读上表定主线结局 → 叙述收束 → `story_over=true`。
- 结局叙述要回扣玩家这一路在四位皇子身上累积的情丝与记忆,让选择有重量。
