# 情丝系统(极简)

这个世界没有战斗、没有掷骰。核心「数值」是每位皇子的**情丝(affection)**,
计时器是**引魂灯的灯火(lamp_remaining)**,玩家的立场记在**心迹(vq 四轴)**里。

## 情丝(affection 0–100,每位皇子各一份)
玩家每一句有分量的话、每一次靠近或回避的选择,都会让对象皇子的情丝变化。
判定交给 `@heart`,它返回 delta(增减),world-agent 把新值写回
`game/npc/<id>.json`。

| 区间 | 等级(affection_level) | 表现 |
|---|---|---|
| 0–19 | 萍水 | 客套、有距离,公事公办 |
| 20–39 | 留意 | 会主动寻话题、记得你说过的小事 |
| 40–59 | 牵念 | 在意你的反应、目光会追着你、替你遮掩 |
| 60–79 | 倾心 | 愿意露出藏着的事(秘密 / 软肋 / 那句话) |
| 80–99 | 情难自禁 | 话到嘴边、险些越礼,却差一点 |
| 100 | 情根深种 | 只要在禁苑引魂灯阁说出真心话,便可定情 |

## 心迹(vq 四轴,记在 run_state.vq,各 −1.0 ~ +1.0)
玩家的立场会在四条轴上累积,决定结局的底色,也决定皇子如何看你:
- `忠诚_背叛`:效忠所爱 +,为利背叛 −。
- `坦诚_欺瞒`:直言以心换心 +,隐瞒算计 −。
- `奉献_索取`:牺牲成全 +,只求回报 −。
- `信任_猜疑`:全然信赖 +,处处提防 −。

heart 每拍可给 `vq_shift`(涉及的轴各 −0.3 ~ +0.3),world-agent 累加进 `run_state.vq`(clamp)。

## 引魂灯的灯火(lamp_remaining 100 → 0)
`run_state.lamp_remaining` 是这段故事的截止线。每推进一夜 / 一场重要场景,
world-agent 让它减少(约每夜 −12~20)。**灯火燃尽(归零)前没有走到「情根深种」并
在灯阁说出口,这段情丝就停在差一点的地方**——这正是这个世界的基调。

## 记忆与回响
每位皇子都有 `memory` 数组。玩家说过的、做过的有分量的事,world-agent 追加进去
(一句话 + 第几夜)。皇子会在很久以后主动提起:「你那夜在凉亭说的那句……」
——**说过的话会回来**;背叛过的人会被猜疑,坦诚过的人会被托付。

## 时间线(timeline)
每次进入故事可能落在不同时间线节点(`first-night` / `after-court` / `lamp-fading` …)。
同样的宫墙,不同的时辰,遇见的人和能说的话都不同。

## 命运抉择(pivot)
本世界有三处命运抉择,在第 2 / 4 / 7 夜由 world-agent 抛出,玩家所选 flag 追加进
`run_state.pivot_flags`。三 flag 集齐后决定八种结局——详见 `endings.md`。

## 给 world-agent 的提示
- 每个有分量的玩家言行 → 调 `@heart` 判定 → 把新情丝 + 四轴心迹 + 这句话写回 npc/run_state。
- 一次只推进**一拍**(一句对话 / 一个瞬间),然后停下,`AskUserQuestion` 给选项。
- 基调由你守住:华丽、暗涌、留白、一点机心,「引魂灯灯火渐弱」不要快进。
