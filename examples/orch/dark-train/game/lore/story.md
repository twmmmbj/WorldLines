# Dark Train — Story Log

> Milestones only. The Keeper appends a section here when a player
> crosses a meaningful threshold — first carriage cleared, first
> fragment recovered, first dispersal, indefinite-insanity onset,
> reaching Carriage 1. Daily wheel-clatter does not belong here.

Format per section:

```
## Day N · Tick M · Carriage X

One paragraph. Short. Lovecraftian. The investigator's perspective.
```

Don't reach for sweeping conclusions; the Train doesn't allow them.
The log is what was learned, what was lost, in that order.
