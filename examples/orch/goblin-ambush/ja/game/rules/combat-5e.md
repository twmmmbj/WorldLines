# D&D 5e 戦闘早見表(超簡易)

この遭遇戦に必要な部分だけをカバーする。

## ターンの流れ
1. **イニシアチブ**:各参戦者が `d20 + イニシアチブ修正値` を振り、高い順に並べる = この戦闘の行動順。
2. 自分の番になったら、**一つの行動**を行う(この戦闘では通常 = 一回の攻撃、または移動/隠れる)。
3. 全員が一回ずつ行動 = 1ラウンド(round)終了、次のラウンドへ、どちらかが倒れるまで。

## 攻撃判定
- `d20 + 命中ボーナス` を振り、**≥ 対象の AC** → 命中。
- **クリティカル**:d20 の出目 = 20 → 必中、かつ**ダメージダイスを倍に**(修正値は倍にしない)。
- **大失敗**:d20 の出目 = 1 → 必中失敗。
- 対象が半身遮蔽を取っているとき AC +2。

## ダメージ
命中後、武器ダメージダイス + 修正値を振り、対象の**現在 HP** から差し引く。
ブリーの急所攻撃は追加で `+2d6`(条件を満たすとき、各ラウンド一回)。

## 倒れる / 終了
- **ゴブリン HP ≤ 0** → 死亡、戦闘終了、勝利。
- **冒険者 HP ≤ 0** → 倒れて気絶、この戦闘から離脱(本 demo は簡略化:死亡セーヴは振らない)。
- ゴブリンの**巧みな逃走**が成功して森へ逃げ込む → 戦闘も終了(逃げられた)。

## DM へのヒント
- ダイスは毎回 `referee` に任せ、結果(命中/外れ、ダメージ値)をそのままプレイヤーに伝え、画面が浮かぶ一文を添える。
- 一度に**一人のキャラクターの行動**だけを進め、止まってプレイヤーを待つ;ゴブリンの番はあなた(DM)が tactics に従って行動させる。
